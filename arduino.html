Arduino Eğitimleri

Bu sayfada temel Arduino projelerini ve derslerini adım adım bulabilirsiniz.

Ders 1: Arduino Nedir?

Arduino; sensörler, butonlar, motorlar gibi donanımlarla çalışmamıza olanak tanıyan açık kaynaklı bir geliştirme platformudur.

Ders 2: LED Yakma

Gerekli Malzemeler:

  • Arduino UNO
  • USB kablo
  • LED
  • 220 ohm direnç
  • Jumper kablolar
  • Breadboard

Bağlantı:

LED'in uzun bacağını Arduino'nun 13 numaralı pinine, kısa bacağını GND'ye bağla. Araya 220 ohm direnç koymayı unutma.

Kod:

void setup() {
  pinMode(13, OUTPUT);
}

void loop() {
  digitalWrite(13, HIGH); // LED yanar
  delay(1000);
  digitalWrite(13, LOW);  // LED söner
  delay(1000);
}

Sonuç:

LED her saniye yanıp söner.

Ders 3: Buton ile LED Kontrolü

Gerekli Malzemeler:

  • 1 x LED
  • 1 x Buton
  • 220 ohm direnç
  • 10k ohm direnç
  • Breadboard ve jumper kablolar

Kod:

const int ledPin = 13;
const int buttonPin = 2;

void setup() {
  pinMode(ledPin, OUTPUT);
  pinMode(buttonPin, INPUT);
}

void loop() {
  int buttonState = digitalRead(buttonPin);
  digitalWrite(ledPin, buttonState);
}

Açıklama:

Butona bastığınızda LED yanar, bıraktığınızda söner.

Yakında:

– Buzzer kullanımı
– LDR ile ışık sensörü
– DHT11 ile sıcaklık/ nem ölçümü
– LCD ekran bağlantısı
– Bluetooth ile kontrol
